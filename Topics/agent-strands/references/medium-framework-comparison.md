## Source
- **Author**: Dr Alexandra Posoldova
- **Platform**: Medium

## Summary
LangGraph, CrewAI, AutoGen, Strands Agents の4フレームワークを実際に使用して比較した記事。学習曲線、本番対応度、使い分けの指針を提供。

## Key Points
- Strands は最も簡単に始められる
- LangGraph はグラフ思考が必要で学習曲線が高い
- CrewAI は直感的だがエンタープライズ寄り
- AutoGen は研究・プロトタイピング向け
