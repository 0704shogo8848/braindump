# GW プリミティブキャンプ — サバイバル生活体験

## Overview

2026年ゴールデンウィーク（4/29〜5/6）に実行する原始的キャンプ体験のリサーチ。森や川で渓流釣りをし、自分で捌いて食べる。火起こし・テント泊・野営を基本とし、裸で過ごしても問題ないレベルの人目のない秘境を候補地とする。農業体験・農塾での自給自足的な暮らしも組み合わせ、都市生活から完全に離れたプリミティブライフを設計する。

## Key Questions

### 場所選定
- GW期間に渓流釣り＋野営が可能で、人がほぼ来ない秘境的スポットはどこか？（関東〜中部〜東北）
- 裸で過ごしても問題ないレベルのプライバシーが確保できる場所の条件と具体的候補地は？
- 私有地・国有林・河川敷など、法的に野営が許可される／黙認されるエリアはどこか？

### 渓流釣り
- GW時期（4月末〜5月初旬）に釣れる渓流魚の種類と狙いやすいポイントは？
- 遊漁券・入漁券の取得方法、費用、注意点は？
- 釣った魚を現地で捌いて食べるための最低限の道具と技術は？

### 火起こし・野営技術
- 摩擦式・フェロセリウムロッドなど、マッチ/ライターなしの火起こし方法と習得のコツは？
- GW時期の山間部の気温・天候を踏まえた野営装備（テント・タープ・寝袋）の選定は？
- 焚き火の法的制約（森林法・消防法）と安全な場所の条件は？

### 食・自給自足
- 渓流魚以外に現地調達可能な食材（山菜・野草・きのこ）でGW時期に採れるものは？
- 最低限持ち込むべき調味料・食材と、完全現地調達はどこまで現実的か？
- 農塾・ファームステイでGW期間に短期体験を受け入れている施設はどこか？

### ロジスティクス
