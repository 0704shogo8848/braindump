# 東北エリアWWOOFホスト候補 — 和賀川源流キャンプとの連携

## 2026年GWカレンダー

- 4/29（水）昭和の日 — 単独祝日
- 5/2（土）〜5/6（水・振替休日）— **5連休確定**
- 有給2日取得 → 4/29〜5/6の**8連休**

**WWOOFは最低2〜3泊の滞在が一般的。** GW前半にWWOOF、後半に和賀川源流キャンプという組み合わせが理想的。

| 項目 | 内容 |
|------|------|
| 初年度費用 | 5,500円（18ヶ月有効・6ヶ月ボーナス） |

| 5年目以降 | 1,500円 |

| 支払方法 | 郵便振替 / PayPay銀行 / PayPal・クレジットカード |

### 登録の流れ

1. wwoofjapan.com で申し込みフォーム送信
2. 登録料を送金
3. 数日で手続き完了 → すぐにホストにメッセージ可能
4. ホストプロフィールを読み、メッセージを送る
5. **重要**: ホストから受入承諾の回答をもらう前に「つながりリクエスト」をクリックしてはいけない（断られる原因になる）

### 東北のホスト傾向

- 日本全国で約300ヶ所のホスト登録あり
- 東北6県（青森・岩手・宮城・秋田・山形・福島）に分類されたリスト
- 有機農家が中心。自給自足度の高いホストも存在
- WWOOF Japanは会員制のためホスト詳細は登録後にのみ閲覧可能
- 各ホストのプロフィールで「受入可能時期」「活動内容」「ブログ」「過去のウーファーの感想」が確認できる

## 3. WWOOF候補（登録後に確認が必要）

WWOOF Japanは会員制のため、個別ホスト名は公式サイト登録後にのみ確認可能。以下は公開情報から推定される有力候補エリア。

### 岩手県で探すべきエリア

#### 花巻市東和町周辺
- **自然農園ウレシパモシリ**（酒勾徹さん）が存在するエリア
  - 1995年開設、パーマカルチャー実践農園
  - 無農薬・無化学肥料の自然栽培（米・麦・雑穀・野菜・果樹）
  - 平飼い養鶏、放牧養豚
  - 研修生の受け入れ実績あり（全国から研修生が来る）
  - 農家民宿のオープンも検討中との情報あり
  - **WWOOFホストかどうかは要確認**（ポケットマルシェで販売中 → 連絡可能）
  - 西和賀町まで車で約60〜70分

#### 北上市周辺
- MOA自然農法北上普及会が活動
- 農業・商業・工業がバランスよく共存する地域
- WWOOFホストの有無は要確認

#### 遠野市
- 農家民宿が複数あり、農泊文化が根付いている
- 西和賀町からはやや遠い（車90分）

### 秋田県で探すべきエリア

#### 仙北市
- グリーンツーリズムの先進地
- 農家民宿が多数あり、外国人受入実績も豊富
- 西和賀町まで横手経由で50〜60分

#### 横手市・大仙市
- 秋田自動車道で西和賀町に直結
- 有機農業実践者がいるエリア
- 大仙市には農家民宿あり

### WWOOFサイトでの検索手順

1. wwoofjapan.com で会員登録（5,500円）
2. ホスト一覧 → 「東北」→「岩手県」または「秋田県」でフィルタ
3. 各ホストプロフィールで以下をチェック:
   - 「受入可能時期」にGW（4月下旬〜5月上旬）が含まれるか
   - 「有機農業」「自給自足」「オフグリッド」「パーマカルチャー」等のキーワード
   - 過去のウーファーのフィードバック
   - ホストのブログ更新頻度（アクティブかどうか）
4. 気になるホストにメッセージ送信
5. 返信なし2〜3週間後に次のホストへ

## 5. プラットフォーム別の探し方

### WWOOF Japan（wwoofjapan.com）
- **費用**: 年会費5,500円（初年度）
- **探し方**: 会員登録 → 県別ホストリスト → プロフィール閲覧 → メッセージ
- **特徴**: 宿泊・食事無料（1日6時間程度の農作業）
- **GW向け**: 東北は農繁期前。田植え準備、山菜収穫、畑の整備など

### STAY JAPAN（stayjapan.com）
- **費用**: 宿泊費のみ（予約手数料あり）
- **探し方**: エリア → 岩手/秋田 → 農泊フィルター
- **特徴**: オンライン即予約可。農業体験オプション付き
- **岩手**: stayjapan.com/area/iwate
- **秋田**: stayjapan.com/area/akita

### SMOUT（smout.jp）
- **費用**: 無料登録
- **探し方**: 農泊特集ページ or 地域検索 → 岩手/秋田
- **特徴**: 移住体験ツアー・農業体験プロジェクトが豊富。お試し住宅情報も
- **注目**: 岩手県雫石町で「狩猟体験」プロジェクトあり

### 農泊ポータル（nohaku.net）
- **探し方**: エリア → 岩手 or 秋田
- **特徴**: 農林水産省推進の公式ポータル。体験+宿泊のセット情報

### 秋田グリーンツーリズム（akita-gt.org）
- **探し方**: 体験 or 泊まる → エリア別
- **特徴**: 秋田県公式。仙北市に豊富な農家民宿リスト
- **問い合わせ**: 仙北市農山村体験推進協議会

### 仙北市グリーンツーリズム（semboku-gt.jp）
- **探し方**: 宿泊/体験メニュー
- **特徴**: 仙北市特化。農家民宿、体験プログラム、アクセス情報が充実

## 9. 注意事項

- GWの東北は**まだ寒い**（朝晩は5℃前後になることも）。防寒必須
- 西和賀町は**豪雪地帯**。4月下旬〜5月上旬は残雪の可能性あり。源流域の道路状況を事前確認すること
- WWOOFはあくまで「労働と食事・宿泊の交換」。観光目的ではないことをホストに伝える
- 農家民宿は**個人経営**が多いため、キャンセルポリシーを事前確認
- 車がないと東北の移動は厳しい。レンタカーの手配も早めに
