# 渓の翁おすすめ！源流・川泊まりのノウハウ～イワナのさばき方編～

**URL**: https://web.tsuribito.co.jp/beginner/genryu-kawadomari-04
**発行元**: つり人社

**概要**: 源流釣りの達人「渓の翁」によるイワナの現地処理方法の詳細ガイド。

**主要ポイント**:
- 水で洗わないことが最重要（旨味が逃げる）
- 肛門からエラ下まで腹を割き、内臓を指で取り出す
- 日本手拭いで清掃（水洗い厳禁）
- 皮むきは歯で押さえて手で剥がす独特の技法
- 28cm程度のイワナ1尾で刺身一人分
- 残った頭・皮・中骨は焼き枯らしにして保存食化

**リサーチとの関連**: 渓流魚を現地で捌いて食べるための実践的な技術ガイドとして直接参照。サバイバルキャンプでの食事の核心部分。
