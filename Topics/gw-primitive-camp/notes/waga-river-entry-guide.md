# 和賀川源流 実践入渓ガイド

## 概要

和賀川は北上川水系の一級河川、全長約80km。奥羽山脈の和賀岳（1,439m）東麓に源を発し、真昼山地東麓の川舟断層に沿って南流、西和賀町川尻で鬼ヶ瀬川と合流した後に東流し、北上市黒沢尻付近で北上川に注ぐ。上流から中流にかけては日本有数の豪雪地帯で、ブナの原生林に覆われた源流域はイワナの楽園として源流釣り師に知られる。

月刊つり人が選ぶ「1泊で行ける源流ベスト5」に選出されており、難所がなくビギナーでも安全に遡行できる渓として評価されている。

## 入渓ルートの詳細

### ルート概要

高下登山口 → 前山越え → 和賀川徒渉点（入渓点）

登山道は和賀岳を目指す道で、支流の高下川沿いに設けられている。この登山道を使い、ひとつ山を隔てた先の和賀川本流に降りる。登山道のため迷うことはない。

### コース数値

| 項目 | 数値 |
|------|------|
| 登山口標高 | 約500m |
| 前山ピーク標高 | 約900m |
| 和賀川徒渉点標高 | 約700m |
| 総標高差（登り） | 約400m |
| 総標高差（下り） | 約200m |
| 登山口→徒渉点 所要時間 | 2〜3時間（個人差あり） |
| 難易度 | 中級（体力は必要だが技術的困難はない） |

### ルート詳細

1. **高下登山口（526m）→ 高下林道終点**
   - 林道歩き。車は登山口まで
   - 道標あり

2. **林道終点 → 前山（約900m）**
   - ブナ林の中の登り
   - 標高差約400m。急登箇所あり
   - 所要約1.5〜2時間

3. **前山 → 和賀川徒渉点（約700m）**
   - 前山から約200m下る
   - 沢筋に向かって降りる
   - 所要約30分〜1時間

4. **徒渉点から源流域**
   - 徒渉点が入渓ポイント
   - ここから上流に向かって遡行開始
   - 渓相はなだらかで難所なし

## テント設営に適した場所

### 源流域の河原（徒渉点〜上流部）

和賀川源流部は比較的穏やかな渓相で、遡行を進めると適度な広さの河原が点在する。

**テント設営の目安**:
- 徒渉点付近から上流に向かい、開けた河原を探す
- 源流部は砂利混じりの河原が多く、テントの設営は比較的容易
- 増水に備え、河原の中でもやや高い位置を選ぶ
- ブナ林の平坦地も候補になるが、クマの行動圏内であることを意識する

**注意事項**:
- タープの方がテントより適している（源流泊の定石）
- 壁がないため増水の察知が容易
- 地形に影響されず設営可能
- 雨天時は河原から離れ、高台のブナ林内に移動すること
- 上流の豪雪地帯からの雪解け水で急な増水のリスクがある

## 水場

- **和賀川本流**: そのまま飲用可能な清冽な水（源流域は基本的に浄水不要だが、念のため浄水器推奨）
- **支流・沢の合流点**: 随所にある小沢が天然の水場になる
- **登山道上**: 前山越えの登山道には明確な水場の表示なし。水は入渓前に確保しておくこと

## GW時期の残雪状況・渡渉の水量

### 残雪（5月上旬）

**西和賀町は日本有数の豪雪地帯**。冬季の積雪は平地でも2m超になる。

- **林道**: 除雪なし。降雪期（11月下旬）から雪解け（5月下旬）まで通行不可
- **高下林道**: 2026年現在、国有林内で災害復旧工事中のため通行止め（要最新確認）
- **高畑林道**: 5月中旬までは積雪あり。5月下旬の雪解けまで通行不可の可能性が高い
- **登山道**: 標高800m以上は残雪あり。前山越えで雪上歩行の可能性が高い
- **源流域**: 6月まで残雪が見られるとの報告あり

### GW入渓の現実的判断

**GW（4/29〜5/6）に和賀川源流入渓は非常に困難の可能性が高い。**

理由:
1. 高下林道が5月下旬まで積雪で通行不可
2. 災害復旧工事による通行止めの可能性
3. 標高800m以上の登山道に残雪（アイゼン・ピッケル必要レベル）
4. 雪解け増水で源流部の水量が多く、徒渉が危険

### 渡渉の水量

- GW時期は雪解け最盛期で、和賀川の水量は年間で最も多い時期の一つ
- 徒渉点（標高700m付近）では膝上〜腰まで浸かる可能性がある
- 渡渉ロープの携行を推奨
- 朝の水温は極めて低い（5℃以下）。ウェーダー必須

### 推奨時期

**6月中旬〜7月**: 雪解け完了、林道開通、水量安定。源流釣りの本番。
**5月下旬**: 林道開通直後。残雪が残るが入渓可能になる年が多い。

## 安全情報

### クマ対策

和賀山塊はツキノワグマの高密度生息域。

- クマ鈴必携
- クマスプレー携行推奨
- 食料はテント/タープから離して吊るす（ベアハング）
- 源流部では渓流の音でクマの気配が消されることがある
- 夕方〜早朝は特に注意

### 単独行動のリスク

- 携帯圏外のため救助要請が困難
- 最低2人以上での入渓を推奨
- 単独の場合は衛星通信デバイス必須
- 行動予定を必ず家族・知人に伝えること

### 増水

- 上流の豪雪地帯からの雪解け水により急な増水がある
- 天候悪化時は即座に河原から離れること
- テント/タープは常に高い場所に設営

## GW実行に向けた判断フロー

```
1. 入山2週間前: 北上土木センターに林道状況を電話確認
   ├── 林道開通 → 計画実行
   └── 林道通行止め → 以下を検討
       ├── 林道徒歩アプローチ（+数時間）で入渓可能か？
       ├── 高畑林道経由の代替ルートは？
       └── 候補地を変更（奥秩父・奥利根等）

2. 入山3日前: 天気予報チェック
   ├── 晴れ〜曇り → 実行
   └── 雨予報 → 増水リスクのため延期検討

3. 入山当日: スーパーオセンで食料調達 → 高下登山口へ
```
