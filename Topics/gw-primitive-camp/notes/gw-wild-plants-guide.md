# GW山菜図鑑 — 食べられるもの vs 致命的な毒草

> **警告**: 山菜採りは命に関わる。確実に同定できない植物は絶対に採らない・食べない。本資料はあくまで参考であり、初心者は必ず経験者と同行すること。

---

## 1. GW時期（4月末〜5月初旬）に東北〜関東で採れる食用山菜

### 1-1. タラの芽（タラノメ）

| 項目 | 内容 |
|------|------|
| **別名** | 山菜の王様 |
| **旬** | 4月〜5月（関東は4月中旬〜、東北は5月上旬〜） |
| **見た目** | トゲだらけの枝先に出る若芽。芽は淡い黄緑色で、開くと掌状複葉になる |
| **生育場所** | 日当たりの良い林道脇、伐採跡地、道路沿い。先駆樹種で明るい場所を好む |
| **採取のコツ** | 一番芽（頂芽）のみ採取し、二番芽以降は翌年のために残す。トゲに注意、革手袋推奨 |
| **下処理** | 根元の硬い部分とハカマを除去。アクは少ないのでそのまま調理可 |
| **調理法** | 天ぷら（最高）、素揚げ、和え物、パスタの具 |
| **栄養価** | タンパク質が山菜中トップクラス。ビタミンE 2.4mg、ビタミンK 99ug、葉酸 160ug、カリウム 460mg、食物繊維 4.2g（100gあたり） |
| **注意** | ヤマウルシの新芽と誤認しやすい（かぶれる）。タラノキはトゲがある、ウルシにはトゲがない |

### 1-2. コシアブラ

| 項目 | 内容 |
|------|------|
| **別名** | 山菜の女王 |
| **旬** | 4月下旬〜5月中旬 |
| **見た目** | 筆のようにまとまった若芽（葉筆）。5枚の掌状複葉。幹は灰白色で滑らか |
| **生育場所** | ブナ帯の落葉広葉樹林内。標高300〜900m程度。やや日陰の斜面 |
| **採取のコツ** | タラの芽同様、頂芽のみ採取。小さすぎる木からは採らない |
| **下処理** | ハカマを取り除く。アクがやや強いので水にさらすか、さっと茹でる |
| **調理法** | 天ぷら（最高）、混ぜご飯、パスタ、和え物 |
| **栄養価** | ケンフェロール、ケルセチン配糖体を含有。抗酸化・抗炎症・血圧低下作用が期待される |
| **注意** | ウルシ科ではないがウコギ科。タカノツメとの混同あり（タカノツメは3枚葉、コシアブラは5枚葉） |

### 1-3. ワラビ

| 項目 | 内容 |
|------|------|
| **旬** | 4月下旬〜5月（関東）、5月〜6月上旬（東北） |
| **見た目** | 拳を握ったような渦巻き状の若芽。茎は緑〜紫色で産毛あり |
| **生育場所** | 日当たりの良い草地、山の斜面、スキー場跡地。群生することが多い |
| **採取のコツ** | 手で折れる柔らかい部分だけ採る。小さすぎるものは残す |
| **下処理** | **あく抜き必須**。重曹または木灰を入れた熱湯に一晩漬ける。プタキロサイド（発がん性物質）を除去するため |
| **調理法** | おひたし、煮物、ナムル、炒め物、山菜蕎麦 |
| **栄養価** | 食物繊維豊富。ビタミンB2、カリウムを含む |
| **注意** | 生食厳禁。あく抜き不十分だと中毒リスクあり。ビタミンB1分解酵素（チアミナーゼ）を含有 |

### 1-4. ゼンマイ

| 項目 | 内容 |
|------|------|
| **旬** | 4月下旬〜5月中旬 |
| **見た目** | 渦巻き状の若芽で綿毛に覆われている。ワラビより太い |
| **生育場所** | 沢沿い、湿った急斜面、崖の近く。採取難易度高め |
| **採取のコツ** | 綿毛を取ってから調理。「男ゼンマイ」（胞子葉）は採らずに残す。「女ゼンマイ」（栄養葉）を採る |
| **下処理** | 綿毛を除去後、茹でて数日かけて揉みながら天日干し（乾燥ゼンマイ）。または重曹であく抜き |
| **調理法** | 煮物、ナムル（韓国風）、炒め物、ビビンバの具 |
| **栄養価** | 食物繊維が非常に豊富。カリウム、鉄分を含む |

### 1-5. コゴミ（クサソテツ）

| 項目 | 内容 |
|------|------|
| **旬** | 4月下旬〜5月上旬 |
| **見た目** | 渦巻き状の若芽だがワラビより太く鮮やかな緑色。株から複数本立ち上がる |
| **生育場所** | 沢沿い、川べりの湿った場所。群生する |
| **採取のコツ** | アクがほぼなく最も初心者向けの山菜。1株から全部採らず数本残す |
| **下処理** | さっと茹でるだけでOK。あく抜き不要 |
| **調理法** | おひたし、天ぷら、サラダ、胡麻和え、マヨネーズ和え |
| **栄養価** | ビタミンA、ビタミンC、食物繊維が豊富 |

### 1-6. ウド（山ウド）

| 項目 | 内容 |
|------|------|
| **旬** | 4月〜5月初旬 |
| **見た目** | 太い茎に産毛。高さ30〜50cm程度の若芽を採る。独特の芳香 |
| **生育場所** | 山間部の斜面、林縁、沢沿い |
| **採取のコツ** | 根を傷めないよう刃物で根元から切る。引き抜かない |
| **下処理** | 皮をむいて酢水にさらす（アク抜き）。皮はきんぴらに |
| **調理法** | 酢味噌和え、天ぷら、きんぴら（皮）、サラダ、味噌汁 |
| **栄養価** | アスパラギン酸（疲労回復）、カリウム（むくみ解消）、食物繊維 |

### 1-7. ギョウジャニンニク（行者ニンニク）

| 項目 | 内容 |
|------|------|
| **旬** | 4月下旬〜5月（東北） |
| **見た目** | 幅広の葉が1〜3枚。茎の根元に赤紫色の薄皮（繊維）がある。強烈なニンニク臭 |
| **生育場所** | 深い山の湿った林床。ブナ帯の日陰 |
| **採取のコツ** | 成長が非常に遅い（種から収穫まで5〜7年）ため、根を残して葉だけ切る。採りすぎ厳禁 |
| **下処理** | 特に不要。さっと洗うだけ |
| **調理法** | 醤油漬け、炒め物、卵とじ、ジンギスカンの付け合わせ、餃子の具 |
| **栄養価** | アリシン（殺菌・疲労回復）、ビタミンA、ビタミンC、セレン |
| **注意** | **最も誤食死亡事故が多い山菜の一つ**。イヌサフラン・スズラン・バイケイソウとの混同に注意 |

### 1-8. ウルイ（オオバギボウシ）

| 項目 | 内容 |
|------|------|
| **旬** | 4月下旬〜5月 |
| **見た目** | 薄緑〜黄緑色の丸まった葉が地面から立ち上がる。葉面はつるつる滑らか |
| **生育場所** | 山地の湿った場所、沢沿い |
| **採取のコツ** | 葉が展開する前の巻いた状態が食べ頃 |
| **下処理** | アクが少ないのでさっと茹でるだけ |
| **調理法** | おひたし、サラダ、浅漬け、鍋物、酢味噌和え |
| **栄養価** | ビタミンC、食物繊維 |
| **注意** | **バイケイソウとの誤食事故が頻発**。葉脈の走り方で判別可能（後述） |

### 1-9. セリ

| 項目 | 内容 |
|------|------|
| **旬** | 3月〜5月 |
| **見た目** | 春の七草の一つ。三つ葉に似た葉で独特の爽やかな香り |
| **生育場所** | 水辺、田んぼの畦、湿地 |
| **採取のコツ** | 根ごと引き抜かず、茎を切る |
| **下処理** | よく洗って泥を落とす |
| **調理法** | セリ鍋（仙台名物）、和え物、天ぷら、おひたし |
| **栄養価** | ビタミンC、ビタミンA、鉄分、カリウム |
| **注意** | **ドクゼリとの誤食は致命的**。根と香りで判別（後述） |

### 1-10. フキノトウ

| 項目 | 内容 |
|------|------|
| **旬** | 東北では4月〜5月上旬（残雪がある山間部） |
| **見た目** | 丸い蕾状で白い綿毛が密生。独特のほろ苦い香り |
| **生育場所** | 雪解け直後の斜面、日当たりの良い山裾 |
| **採取のコツ** | 花が開く前の蕾状態が食べ頃。開いたものは苦みが強い |
| **下処理** | 外側の皮を剥いて水にさらす。フキノトウ味噌にする場合は刻んで炒める |
| **調理法** | 天ぷら、フキノトウ味噌、味噌汁、パスタ |
| **栄養価** | ビタミンK、葉酸、カリウム、フキノール酸（抗アレルギー） |
| **注意** | **根には毒（ピロリジジンアルカロイド）がある**。根は食べない。ハシリドコロ・フクジュソウとの誤認に注意 |

## 2. 致命的な毒草 — 山菜と誤認しやすい猛毒植物一覧

### 2-1. トリカブト（附子）

| 項目 | 内容 |
|------|------|
| **誤認される山菜** | ニリンソウ、モミジガサ（シドケ）、ゲンノショウコ |
| **毒成分** | アコニチン、メサコニチン、ヒパコニチン（アルカロイド） |
| **毒の強さ** | **日本三大有毒植物の一つ**。致死量はアコニチンとして2〜6mg（ほんの微量で死に至る） |
| **有毒部位** | 全草。毒性の強さは 根（塊根）> 葉 > 茎 |
| **症状** | 食後10〜20分で発症。口唇・舌のしびれ → 手足のしびれ → 嘔吐・腹痛・下痢 → 不整脈・血圧低下 → 痙攣・呼吸不全 → 死亡 |
| **死亡までの時間** | 6時間以内。24時間以上生存すれば回復傾向 |
| **応急処置** | 口に入れてピリピリしたら即吐き出す。既に飲み込んだ場合は催吐→大量の水・茶・牛乳を飲む→直ちに救急車。**解毒剤・特効薬は存在しない** |

### 2-2. イヌサフラン（コルチカム）

| 項目 | 内容 |
|------|------|
| **誤認される山菜** | ギョウジャニンニク、ウルイ |
| **毒成分** | コルヒチン |
| **毒の強さ** | 致死量はコルヒチンとして約4.3mg（体重50kgの場合） |
| **有毒部位** | 全草（球根、葉、花すべて）。種子に0.2〜0.6%、鱗茎に0.08〜0.2%含有 |
| **症状** | 嘔吐、下痢、皮膚の知覚減退、呼吸困難。重症で死亡 |
| **応急処置** | 催吐し直ちに医療機関へ。特効薬なし |

### 2-3. バイケイソウ（梅蕙草）

| 項目 | 内容 |
|------|------|
| **誤認される山菜** | ウルイ（オオバギボウシ）、ギョウジャニンニク |
| **毒成分** | プロトベラトリン、ジェルビン、シクロパミン、ベラトラミン（アルカロイド） |
| **毒の強さ** | **加熱しても毒性は消えない**。死亡事例あり |
| **有毒部位** | 全草（新芽・葉・茎・根すべて） |
| **症状** | 食後30分〜1時間で発症。嘔吐、下痢、血圧低下、心拍数低下、めまい、手足のしびれ、痙攣。重症で意識不明→死亡 |
| **応急処置** | 催吐し直ちに医療機関へ |

### 2-4. ドクゼリ（毒芹）

| 項目 | 内容 |
|------|------|
| **誤認される山菜** | セリ |
| **毒成分** | シクトキシン（ポリアセチレン系猛毒） |
| **毒の強さ** | **日本三大有毒植物の一つ**。少量で致死 |
| **有毒部位** | 全草。特に根茎に多い |
| **症状** | 嘔吐、下痢、腹痛、めまい、痙攣、呼吸困難→死亡 |
| **応急処置** | 催吐し直ちに救急車 |

### 2-5. ハシリドコロ

| 項目 | 内容 |
|------|------|
| **誤認される山菜** | フキノトウ |
| **毒成分** | トロパンアルカロイド（ヒヨスチアミン、スコポラミン、アトロピン） |
| **毒の強さ** | 全草有毒。根茎と根が特に強毒 |
| **有毒部位** | 全草 |
| **症状** | 幻覚・錯乱状態（「走り回る」のが名前の由来）、嘔吐、下痢、血便、瞳孔散大、めまい、異常興奮。多食で呼吸困難→死亡 |
| **応急処置** | 催吐し直ちに医療機関へ |

### 2-6. スズラン

| 項目 | 内容 |
|------|------|
| **誤認される山菜** | ギョウジャニンニク |
| **毒成分** | コンバラトキシン、コンバラマリン、コンバロシド（強心配糖体） |
| **毒の強さ** | 全草有毒。花と根に特に多い。花を活けた水でも中毒する |
| **有毒部位** | 全草 |
| **症状** | 摂取後1時間以内に発症。嘔吐、頭痛、めまい、血圧低下、心臓麻痺→死亡 |
| **応急処置** | 催吐し直ちに医療機関へ |

### 2-7. フクジュソウ（福寿草）

| 項目 | 内容 |
|------|------|
| **誤認される山菜** | フキノトウ |
| **毒成分** | シマリン、アドニトキシン（強心配糖体） |
| **有毒部位** | 全草（特に根と茎） |
| **症状** | 嘔吐、下痢、不整脈、心停止 |
| **応急処置** | 催吐し直ちに医療機関へ |

## 3. 見分け比較表 — 命を守る識別ポイント

### 3-1. ニリンソウ（食） vs トリカブト（致死）

| 識別ポイント | ニリンソウ（食用） | トリカブト（猛毒） |
|-------------|-------------------|-------------------|
| **葉の形** | 3裂する掌状の葉。切れ込みが浅め | 同じく3〜5裂するが、切れ込みが深い |
| **花** | 春に白い花（2輪） | 夏〜秋に紫色の兜形の花 |
| **茎** | 細く柔らかい | やや太く硬い |
| **生育** | 群生する。トリカブトと混生することが多い | ニリンソウの群落に混じって生えることがある |
| **根** | 細い根茎 | 太い塊根（ここに最も毒が多い） |
| **最大の問題** | **花が咲く前の若芽の段階では、プロでも葉だけでの区別が極めて困難** |
| **対策** | 花が咲いている株の近くから採る。1本ずつ慎重に確認。少しでも怪しければ採らない |

### 3-2. ギョウジャニンニク（食） vs イヌサフラン（致死）

| 識別ポイント | ギョウジャニンニク（食用） | イヌサフラン（猛毒） |
|-------------|--------------------------|---------------------|
| **匂い** | **葉をちぎると強烈なニンニク臭**（最重要） | 匂いなし |
| **根元の色** | 赤紫色の薄皮（繊維状） | 緑色。紫色の薄皮なし |
| **根の形** | ひげ状の白い根 | 球根（丸い塊） |
| **葉の枚数** | 1〜3枚が1つの茎から出る | 複数枚が密に出る |
| **葉幅** | やや細長い | ギョウジャニンニクより幅広 |

### 3-3. ギョウジャニンニク（食） vs スズラン（致死）

| 識別ポイント | ギョウジャニンニク（食用） | スズラン（猛毒） |
|-------------|--------------------------|-----------------|
| **匂い** | **強烈なニンニク臭**（最重要） | 匂いなし |
| **葉の枚数** | 1〜3枚 | 2枚が基本で重なるように出る |
| **茎の根元** | 赤紫色の薄皮あり | 薄皮なし |
| **葉の質感** | やや厚みがある | やや硬く光沢がある |

### 3-4. ウルイ（食） vs バイケイソウ（致死）

| 識別ポイント | ウルイ/オオバギボウシ（食用） | バイケイソウ（猛毒） |
|-------------|------------------------------|---------------------|
| **葉脈** | **中央に太い主脈があり、左右にカーブして走る**（最重要） | 主脈がなく、付け根から平行に走る |
| **葉の質感** | つるつる滑らか | ざらざらしている |
| **葉柄** | 長い葉柄があり、地際から生える | 葉柄がなく、茎に直接つく |
| **茎** | 茎がない（根生葉） | 茎が立ち上がり、葉が交互につく |
| **若芽** | 巻き方が筒状 | 巻き方がやや扁平 |

### 3-5. セリ（食） vs ドクゼリ（致死）

| 識別ポイント | セリ（食用） | ドクゼリ（猛毒） |
|-------------|-------------|-----------------|
| **匂い** | **茎をちぎると爽やかなセリの香り**（最重要） | ほとんど匂いがない |
| **根** | 白く細いひげ根 | **太い根茎にタケノコ状の節がある**（最重要） |
| **茎の中** | 下部にわずかな空洞、上部は詰まっている | 中空 |
| **大きさ** | 草丈40cm以下 | 草丈1m以上になる大型 |
| **生育場所** | 田んぼの畦、小川 | 湿地、沼地 |

### 3-6. フキノトウ（食） vs ハシリドコロ（猛毒）

| 識別ポイント | フキノトウ（食用） | ハシリドコロ（猛毒） |
|-------------|-------------------|---------------------|
| **匂い** | 独特のフキの香り | 弱い香り |
| **表面** | **白い綿毛が密生**（最重要） | 毛がなく光沢がある |
| **中身** | 中に多数の蕾が詰まっている | 葉が重なるだけ |
| **根** | 地下茎につながる | 直接根につながる |
| **色** | 淡い黄緑色 | やや暗い緑色〜褐色 |

### 3-7. フキノトウ（食） vs フクジュソウ（猛毒）

| 識別ポイント | フキノトウ（食用） | フクジュソウ（猛毒） |
|-------------|-------------------|---------------------|
| **表面** | 白い綿毛が密生 | 毛がなく光沢がある |
| **蕾の形** | 丸く包まれた蕾 | 細い葉が密に重なる |
| **花** | 黄色い小花が多数 | 大きな黄色い花弁 |

## 4. 特に危険な組み合わせワースト5

| 順位 | 食用 | 猛毒 | 危険度 | 理由 |
|------|------|------|--------|------|
| **1** | ニリンソウ | トリカブト | **致死** | 若芽の段階で見た目がほぼ同一。混生する。プロでも判別困難。2026年4月に北海道で80代男性が死亡 |
| **2** | ギョウジャニンニク | イヌサフラン | **致死** | 葉の形が酷似。匂いで判別できるが「確認せず複数本まとめて採る」ケースで事故発生。死亡事例多数 |
| **3** | ウルイ | バイケイソウ | **致死** | 若芽の段階で非常に酷似。宮城県で食中毒が頻発。死亡事例あり |
| **4** | セリ | ドクゼリ | **致死** | 水辺に生え、葉の形が似る。根を確認しないと判別困難 |
| **5** | ギョウジャニンニク | スズラン | **致死** | 葉の形が類似。庭植えのスズランを山菜と間違える事故も |

## 5. 誤食時の緊急対応フロー

```
1. 異変を感じたら（しびれ、ピリピリ感、吐き気など）
   ↓
2. 口の中のものを全て吐き出す
   ↓
3. 119番（救急車）に電話
   ↓
4. 可能なら催吐（指を喉に入れて吐かせる）
   ※ 意識がない人には催吐させない（誤嚥の危険）
   ↓
5. 大量の水・茶・牛乳を飲む（毒の吸収を遅らせる）
   ↓
6. 食べた植物の残り・写真を保存（医師に見せるため）
   ↓
7. 病院搬送（胃洗浄・対症療法）
```

**重要**: トリカブト・イヌサフラン・ドクゼリには解毒剤が存在しない。時間との勝負。

## 6. 山菜採りの基本ルール

### 6-1. 採取のマナー

- **入山許可の確認**: 私有地・管理地での無断採取は窃盗罪に該当する可能性がある。自治体・管理者・観光協会に事前確認
- **採りすぎない**: 食べきれる量だけ採る。タラノキ・コシアブラは頂芽（一番芽）だけ。二番芽まで採ると木が枯れる
- **根を傷めない**: フキ・ウドは刃物で切る。引き抜かない
- **小さい株は残す**: ワラビ・ゼンマイの小さいものは採らずに翌年に備える
- **男ゼンマイは残す**: 胞子葉を採ると繁殖できなくなる

### 6-2. 安全対策

- **確信がなければ採らない**: これが唯一にして最大のルール
- **1本ずつ確認する**: 特にニリンソウの群落ではトリカブトが混じる。まとめて採らない
- **匂いを嗅ぐ習慣**: ギョウジャニンニク→ニンニク臭、セリ→爽やかな香り、がなければ毒草を疑う
- **経験者と同行**: 初心者だけで山菜採りに行かない
- **図鑑・写真を持参**: スマホに写真を保存しておく
- **食べた植物を記録**: 万が一の中毒時に医師への情報提供に必要

### 6-3. 装備

- 革手袋（トゲ対策）
- 刃物（ナイフ or 鎌）
- 通気性の良い籠やメッシュバッグ（ビニール袋は蒸れる）
- 熊鈴・ホイッスル
- 長袖・長ズボン（ダニ・ヤマビル対策）
- 携帯電話（電波の確認）

## 7. 山菜の保存方法

### 7-1. 冷凍保存（1ヶ月以内に消費）

| 手順 | 内容 |
|------|------|
| 1 | 下処理（あく抜き等）を完了させる |
| 2 | さっと茹でて冷水で急冷 |
| 3 | しっかり水気を切る |
| 4 | 小分けにしてラップで包み、ジップロックへ |
| 5 | 急速冷凍（金属トレーに載せると早い） |

**向いている山菜**: コゴミ、ウド、フキ、ワラビ（あく抜き済）

### 7-2. 塩漬け（数ヶ月〜1年以上保存可能）

| 手順 | 内容 |
|------|------|
| 1 | 採取後すぐに処理する（鮮度が命） |
| 2 | 汚れを取り除く |
| 3 | 容器（樽・タッパー）の底に粗塩を敷く |
| 4 | 山菜を並べ、上からも塩をたっぷりふる（山菜の重量の30〜40%の塩） |
| 5 | 重しを載せる |
| 6 | 冷暗所で保存 |
| 7 | 使うときは水で塩抜き（半日〜1日） |

**向いている山菜**: ワラビ、フキ、ゼンマイ、ウド

### 7-3. 乾燥保存（軽量・長期保存）

| 手順 | 内容 |
|------|------|
| 1 | 下処理後、さっと茹でる |
| 2 | 天日干しで数日かけて乾燥（時々揉む） |
| 3 | カラカラになるまで干す |
| 4 | 密閉袋・瓶で冷暗所保存 |
| 5 | 使うときは水で戻す（数時間〜一晩） |

**向いている山菜**: ゼンマイ（定番）、ワラビ、コゴミ

### 保存方法の比較

| 方法 | 保存期間 | メリット | デメリット |
|------|---------|---------|-----------|
| 冷凍 | 1ヶ月 | 手軽、食感が比較的残る | 冷凍庫スペースが必要 |
| 塩漬け | 1年以上 | 長期保存、熟成で旨味が増す | 塩抜きに時間がかかる |
| 乾燥 | 1年以上 | 軽量、風味が凝縮する | 干す手間と天候に左右される |
