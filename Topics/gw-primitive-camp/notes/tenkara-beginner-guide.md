# テンカラ入門 — ゼロから初釣果まで

## 1. テンカラ釣りとは

### 概要

テンカラは日本古来の毛バリ釣り。リールを使わず、延べ竿・ライン・毛バリの3点だけで渓流魚（イワナ・ヤマメ・アマゴ）を狙う。ウキもオモリも使わない、究極にシンプルな釣り。

### 歴史

- 「テンカラ」という名称が広まったのは1970〜80年代。それ以前は地域ごとに「毛鉤」「毛釣り」などと呼ばれていた
- イワナ釣りの記録は元禄年間（1688〜1704年）まで遡れる
- 語源・発祥地は不明。山間部の職漁師が実用的な釣法として磨き上げてきた
- 近年は海外でも "Tenkara" として人気が急上昇中

### 他の釣り方との違い

| 項目 | テンカラ | フライフィッシング | エサ釣り |
|------|---------|-------------------|---------|
| リール | 不要 | 必要 | 不要（延べ竿の場合） |
| 毛バリ/エサ | 毛バリのみ | フライ（多種多様） | 生エサ・練りエサ |
| ライン | 竿と同程度の長さ | リールから繰り出す | オモリ+ウキで沈める |
| 手返し | 非常に速い | やや遅い | 待ちの釣り |
| 道具の数 | 最小限（竿+糸+毛バリ） | 多い（リール+各種フライ） | 中程度 |
| 毛バリの考え方 | パターンにこだわらない | マッチザハッチ（虫に合わせる） | — |
| 得意なフィールド | 狭い渓流・源流 | 広い川・湖 | 全般 |

テンカラの最大の特徴は **シンプルさ** と **手返しの速さ**。毛バリをキャストして3秒流し、反応がなければ即回収→再キャスト。ランガンスタイルで魚を探っていく。

## 2. 竿の選び方

- **調子**：6:4（胴調子）か7:3（先調子）。初心者は **7:3** がキャストしやすく、ピンポイントに狙いやすい

## 3. ラインの選び方

### ラインの種類

テンカラのラインは3種類ある。

#### レベルライン（主流・初心者おすすめ）
- フロロカーボンの単糸。太さが均一
- スプールからそのまま引き出して使える手軽さ
- 現在のテンカラで最も主流
- 安価で入手しやすい
- **初心者は 3.5〜4.5号** がおすすめ（太めの方がキャストのタイミングを取りやすい）

#### テーパーライン
- 撚り糸で先端に向かって細くなる
- 重さがあるため **最も振り込みしやすい**（風にも強い）
- ヨレやすいのがデメリット。寿命が短め
- キャストに自信がない完全初心者には実はテーパーもアリ

#### ストレートライン
- 「テンカラ大王」石垣尚男氏監修でフジノが開発
- レベルラインとテーパーラインの中間的存在
- 振り込みやすさと扱いやすさを両立

### ラインの長さ

- 基本は **竿と同じ長さ**（3.3mの竿なら3.3mのライン）
- 初心者は竿より少し短め（−50cm程度）にするとコントロールしやすい

## 4. 毛バリの種類と選び方

### 毛バリの基本3タイプ

**普通毛鉤（ノーマル）**
- ハックル（羽根）が釣り針の軸に対して垂直に伸びる
- 最もオーソドックスな形状

**逆さ毛鉤（さかさけばり）**
- ハックルが針先の方向に向く
- 水の抵抗を受けやすく、ラインが張りやすい
- アタリが明確に出やすい → **初心者に最もおすすめ**

**順毛鉤（じゅんけばり）**
- ハックルが針の曲がりに沿って伸びる
- 水面に浮きやすい（ドライ向き）

### サイズの選び方

テンカラの毛バリは **フックサイズ（番手）** で選ぶ。数字が大きいほど小さい。

| 状況 | サイズ | 用途 |
|------|--------|------|
| 高活性時・増水時 | #12 | 大きめで魚にアピール |
| 通常（GW時期の渓流） | **#14** | 万能サイズ。迷ったらこれ |
| 低活性時・ハイプレッシャー | #16〜#18 | 春先・人が多い場所 |

### GW時期（4月下旬〜5月上旬）に適したパターン

- **#14 の逆さ毛鉤**（クリーム系・ダーク系）が万能
- カゲロウやトビケラの羽化が活発な時期。水生昆虫を意識した毛バリが有効
- GWは釣り人が増えるため、プレッシャーが高い場所では **#16にサイズダウン** すると効果的
- 色はあまりこだわらなくてよい（テンカラの基本思想）。ただし渓流の水色に合わせて明るめ/暗めを使い分けるとベター

### おすすめ毛バリセット

| 商品名 | 内容 | 価格 | 備考 |
|--------|------|------|------|
| ダイワ テンカラ毛針セット スタンダード春 | #12×2本、#14×2本（クリーム系） | 約500〜700円 | GW時期にピッタリ |
| ダイワ テンカラ毛針セット スタンダード夏 | 陸生昆虫パターン | 約500〜700円 | 夏以降に |
| オーナー テンカラ毛鉤セット | 各種アソート | 約400〜800円 | 種類が豊富 |

> **結論**：最初は **ダイワ スタンダード春 #14** を買えばOK。毛バリの種類にこだわるのはフライフィッシングの考え方。テンカラは「何でも釣れる」精神が基本。

## 5. 仕掛けの作り方

### 仕掛けの全体構成

```
竿先（リリアン） → ライン（3〜4m） → ハリス（0.8号 約1m） → 毛バリ
```

たったこれだけ。ウキもオモリも不要。

### 結び方（3つの結びをマスターすればOK）

#### 1. リリアンとライン（チチワ結び）

1. ラインの端を折り返して輪を作る
2. 折り返した部分で8の字結びを作る（端を輪に2回通す）
3. きっちり締め込んで小さなチチワ（輪）を作る
4. リリアンの先端に結びコブを作っておく
5. チチワをリリアンに通し、結びコブで止まるようにセット

#### 2. ラインとハリス（サージェンスノット）

1. ラインとハリスの端を15cmほど重ねて持つ
2. 重ねた2本で輪を作る
3. 輪の中に両方の端を **2回** くぐらせる
4. 4本（ライン本線・ライン端・ハリス本線・ハリス端）を同時にゆっくり引っ張って締める
5. 余った端糸をカット

#### 3. ハリスと毛バリ（クリンチノット）

1. 毛バリのアイ（穴）にハリスを通す
2. 本線に端糸を **4〜5回** 巻き付ける
3. アイの近くにできた最初の輪に端糸を通す
4. さらにその上にできた大きな輪に端糸を通す
5. 本線をゆっくり引っ張って締め込む

> **ポイント**：結び目は必ず **唾で湿らせてから** 締める。摩擦熱でラインが弱くなるのを防ぐ。

### ハリスについて

- 素材：**フロロカーボン 0.8号**
- 長さ：約 **1m**（80cm〜1.2m）
- ナイロンでもOK（伸びがあるのでバラシにくい。初心者向き）

## 6. キャスティング技術

### 基本のキャスト（オーバーヘッドキャスト）

テンカラは **「振りが全て」**。キャスティングさえ決まればビギナーでも釣れる。

#### バックキャスト（後ろ振り）

1. 竿を水平（9時の位置）に構える
2. 手首のスナップで竿を **12時の位置** まで振り上げる（ここで **ピタッと止める**）
3. ラインが後方に伸びるのを感じる
4. 竿が後方に曲がり込む感覚を待つ

**コツ**：力加減は全体の **7割** をバックキャストに使う。しっかり後方にラインを送ることが最重要。

#### フォワードキャスト（前振り）

1. 後方で曲がった竿の反発力を利用する
2. 竿を **10時〜2時の位置** で止める意識
3. 「前に飛ばそう」と力を入れない。竿のしなりに任せる
4. ラインが前方に伸びていき、毛バリがターゲットに着水

**コツ**：力加減は全体の **3割**。竿の反発を「そっと手助けする」感覚。力むと逆にラインが暴れる。

#### リズム

- **「イーッチ、ニ」** という変則的な2拍子
- バックキャストでゆっくり溜めて（イーッチ）、フォワードは素早く（ニ）
- 慣れるまでは後ろを振り返って、ラインが後方に伸びたのを確認してからフォワードに移るとよい

### 応用：ボウ&アロー（弓矢キャスト）

枝が被さった狭い場所で使う技。

1. 毛バリを手で持つ
2. 竿を引き絞って弓のようにしならせる
3. 毛バリを離して反動で狭い隙間に投げ込む

先行者が敬遠する難ポイントを攻略できる武器。

### キャスティングの練習法

1. **芝生や広場で素振り**：水がなくてもラインの動きは確認できる。目標物を置いて精度を上げる
2. **太めのライン（4号以上）** で練習すると、ラインの重さでタイミングが取りやすい
3. **短めのライン（竿−50cm）** から始めてコントロールを掴む
4. 1日30分×3日も練習すれば基本は身につく

## 7. ポイントの読み方（イワナが潜む場所）

### 渓流魚がいる基本条件

魚は **「身を隠せる場所」** の近くで **「エサが流れてくる場所」** にいる。

### 主要ポイント

#### 落ち込み（滝壺）
- 水が上から落ち込んでいる場所。白泡の下に魚が潜む
- 酸素が豊富で水温も安定。大型が潜んでいることも多い
- 毛バリは白泡の脇〜泡が消える境目を流す

#### 淵（ふち）
- 深くなっている場所。大型のイワナが定位しやすい
- 流心脇の緩流帯が狙い目

#### 瀬（せ）
- 浅くて流れが速い場所。意外と魚がいる
- 活性が高い時は瀬の中で積極的にエサを食う
- テンカラの手返しの速さが活きるポイント

#### 石裏・岩陰
- 大きな石の下流側は流れが緩やかになり、魚の休憩場所
- 石に囲まれた流れ込みは **イワナの代表的なポイント**

#### 肩（かた）
- 落ち込みのすぐ上の部分
- 毛バリを下の段に落とすくらいの気持ちでギリギリまで流すと食いつくことが多い

#### 巻き返し・ヨレ
- 流れがぶつかってできる渦。エサが溜まりやすい
- 泡の筋ができている場所はゴミや虫が集まる証拠 → 魚もいる

### イワナ特有の見分け方

- イワナは **上流部・源流域** に多い（標高が高い場所）
- ヤマメ・アマゴより **暗い場所・冷たい水** を好む
- 落差のある石に囲まれた流れ込みが代表的な住処
- 警戒心はヤマメより低め。近づきすぎなければ何度でもアタックしてくる

### 泡の筋を読む

水面に浮かせるテンカラでは **泡の筋** が最大のヒント。泡が流れる筋 = 虫やゴミが流れる筋 = 魚がエサを待っている場所。

## 8. アワセのタイミング

### アタリの取り方

1. **毛バリが見える場合**：魚が水面で毛バリをくわえる瞬間が見える（最高に興奮する瞬間）
2. **毛バリが見えない場合**：ラインの先端に注目。ツツッと引っ張られる・止まる・上流に動くなど、不自然な動きがあれば全てアワセる
3. **水中の光**：魚が反転する際にキラッと光が見えることがある

### アワセの動作

- **手首をシャープに返すだけ**。竿を大きく振り上げる必要はない
- 手首のスナップで十分ハリ掛かりする
- 大振りすると毛バリが飛んでいってしまう

### タイミングの目安

| 状態 | 判定 |
|------|------|
| 口の横（カンヌキ）に掛かっている | タイミング完璧 |
| 毛バリを飲み込んでいる | やや遅い（でもOK） |
| 上顎・唇に掛かっている | やや早い |
| バラシが多い | 早すぎ or ラインがたるんでいる |

### 初心者最大の敵：びっくりアワセ

魚が浮いてくるのが見えると、反射的にバシッと合わせてしまう。これが **早すぎ** の原因。

**対策**：魚が毛バリをくわえて **反転し、水面下に沈んでから** アワセる。「一呼吸置く」を意識。

### ラインテンションの重要性

ラインがたるんでいると、アワセても力が伝わるまでにタイムラグが生じる。常にラインを軽く張った状態を維持することが、確実なフッキングのカギ。

## 9. 渓流での立ち回り方

### 大原則：下流から上流へ釣り上がる

渓流釣りでは **「釣り上がり」** が鉄則。理由は2つ。

1. **魚は上流を向いている**：エサが上流から流れてくるので、魚は頭を上流に向けて定位している。下流側から近づけば、魚の死角からアプローチできる
2. **マナー**：釣り下ると、後からくる釣り人のポイントを潰してしまう。古くからの渓流のルール

### 実際の立ち回り

1. **入渓点を決める**：橋や林道から川に降りる。入渓点から上流に向かって釣り上がる
2. **ポイントの手前で止まる**：良さそうなポイントを見つけたら、いきなり横まで行かない。少し手前（下流側）から毛バリを打つ
3. **手前のポイントから順に攻める**：近い場所→遠い場所の順。奥から攻めると手前の魚を散らしてしまう
4. **1ポイント3投ルール**：同じ筋を3回流して反応がなければ、魚がいないか食い気がないと判断。次のポイントへ移動
5. **足音を立てない**：渓流魚は振動に敏感。ドスドス歩かず、静かに移動する
6. **影を落とさない**：自分の影が水面に落ちると魚が逃げる。太陽の位置を意識して立ち位置を決める

### テンカラのテンポ

- キャスト → **3秒流す** → 反応なし → 即回収 → キャスト
- 粘らない。どんどん新しいポイントを攻めていく
- 1時間で100m〜200m釣り上がるくらいのペースが目安

### 安全上の注意

- ウェーダー（胴長靴）またはウェーディングシューズを履く
- フェルトソールで滑りにくくする
- 増水時は絶対に入渓しない
- 単独行動を避ける。少なくとも行き先を誰かに伝える

### Step 2：自宅・公園でキャスト練習（1〜3日）

1. 竿にラインをセット（ハリスと毛バリは付けない。代わりにラインの先に毛糸を結ぶ）
2. 広い場所で基本のオーバーヘッドキャストを練習
3. 目標物（ペットボトルなど）を5〜8m先に置いて精度を上げる
4. 「後ろ7割、前3割」のリズムを体に覚え込ませる
5. 1日30分でOK。3日間やればかなり安定する

### Step 3：管理釣り場で実釣（初回）

自然渓流の前に、まず管理釣り場（エリアトラウト）で腕試し。

- **ポンドタイプ**（池型）がキャスト練習の延長として最適
- 魚が多いので毛バリへの反応を体験しやすい
- アワセの感覚を掴める
- 「テンカラOK」の管理釣り場を事前に確認すること

### Step 4：渓流デビュー（GW時期）

1. 遊漁券を購入（現地のコンビニ・釣具店、またはフィッシュパス等のアプリ）
2. 入渓点を確認（Google Mapsの航空写真で橋や駐車スペースをチェック）
3. 下流から釣り上がる
4. 手前のポイントから丁寧に。3投ルールで効率的に
5. 最初は魚影の濃い場所を選ぶ（放流直後の区間など）

### Step 5：振り返りと改善

- 釣れた場所・釣れなかった場所を記録する
- キャストの精度、アワセのタイミングを振り返る
- 毛バリのサイズを変えて試す

## 11. よくある失敗と対策

### 1. ラインが木に引っ掛かる

**原因**：バックキャストで後方を確認していない
**対策**：
- キャスト前に必ず後方を確認する
- 頭上に枝がある場所では横振り（サイドキャスト）やボウ&アローを使う
- 予備のラインと毛バリを多めに持参する

### 2. キャストがまっすぐ飛ばない

**原因**：力の入れすぎ / 竿を倒しすぎ
**対策**：
- 竿は12時〜2時の範囲で動かす意識
- 力を抜く。特にフォワードキャストは竿の反発に任せる
- 太めのライン（4号以上）で練習する

### 3. 魚がバレる（掛かっても逃げられる）

**原因**：アワセが早すぎ / ラインがたるんでいる
**対策**：
- 魚が反転してから合わせる（一呼吸置く）
- ラインを常に軽く張った状態にする
- アワセは手首のスナップだけ。大振りしない

### 4. 魚がいるのに反応しない

**原因**：毛バリのサイズが大きすぎ / アプローチで魚を散らしている
**対策**：
- 毛バリを#16〜#18にサイズダウン
- ポイントから距離を取ってキャスト
- 足音を立てない。影を落とさない

### 5. 毛バリが不自然に流れる（ドラグ）

**原因**：ラインが流れに引っ張られて毛バリが不自然な速度で動く
**対策**：
- ラインをできるだけ水面に付けない（竿を高く持つ）
- 不自然な流れになったら即回収してキャストし直す
- 流れの筋に沿ってラインを置く

### 6. 竿を折る

**原因**：竿先を岩や木にぶつける / 魚を無理に引き抜く
**対策**：
- 移動時は竿先を上に向けて持つ
- 魚は竿の弾力で寄せる。引き抜かない
- 最初は安い竿で始める（折っても精神的ダメージが少ない）

### 7. 結び目がほどける

**原因**：締め込みが甘い / 唾で湿らせていない
**対策**：
- 結び目は必ず唾で湿らせてから締める
- 締め込んだ後に端糸を引っ張って確認
- 出発前に自宅で結び方を練習しておく

## 参考情報

- テンカラ釣りの歴史：元禄年間から続く日本の伝統釣法
- 解禁期間：多くの渓流は3月〜9月（地域により異なる。要確認）
- GW時期は解禁中。水温も上がり始めて魚の活性が高くなる好シーズン
- 遊漁券はフィッシュパス（アプリ）で事前購入が便利
