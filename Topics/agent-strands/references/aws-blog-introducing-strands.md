# Introducing Strands Agents, an Open Source AI Agents SDK

## Source
- **Publisher**: AWS Open Source Blog
- **URL**: https://aws.amazon.com/blogs/opensource/introducing-strands-agents-an-open-source-ai-agents-sdk/
- **Date**: 2025年5月

## Summary
Strands Agents の初回公式発表記事。モデル駆動型アプローチの設計思想、3つのコアコンポーネント（Model, Prompt, Tool）、対応 LLM プロバイダー、ツールシステム（デコレータ + MCP）、デプロイオプションを解説。

## Key Points
- モデル駆動型: LLM がワークフローを自律決定
- 3行のコードでエージェントを構築可能
- AWS 内部（Amazon Q, AWS Glue 等）で本番使用
- Apache 2.0 ライセンス

## Relevance
アーキテクチャ・コア概念の調査に直接関連。
