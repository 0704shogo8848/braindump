# イラン戦争

## Overview

イランを巡る軍事紛争・戦争の歴史と現在の地政学的緊張を調査する。イラン・イラク戦争（1980-1988）、米国との対立の歴史、核開発問題、中東地域における代理戦争、そして現在の軍事的緊張関係を包括的にカバーする。
