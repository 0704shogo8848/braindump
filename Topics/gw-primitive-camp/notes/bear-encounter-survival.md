## 3. GW時期（4月末〜5月初旬）のクマの行動パターン

### 冬眠からの覚醒

- **冬眠期間**：11月中旬〜4月下旬（約5ヶ月間）
- **覚醒時期**：4月中旬〜5月上旬に穴から出る
- **出産メスは遅い**：冬眠中に出産した母グマは5月下旬頃まで巣穴にいる
- 覚醒直後は「栓」と呼ばれる大きな糞をするのが特徴

### GW時期の食性

| 食物 | 詳細 |
|------|------|
| 前年の堅果 | 雪崩地に集まり、前年落下のブナ実・ドングリを食べる |
| 山菜・新芽 | ブナの若葉・花、フキノトウ、早春の山菜 |
| 草本類 | 木本の新芽や新葉 |
| 昆虫類 | 朽木の中のアリなど |

**重要**：ツキノワグマは食肉目だが植物食中心の雑食性。植物の消化効率が極めて悪いため、大量に食べて大量に移動する。

### 行動範囲と活動時間

- **行動範囲**：オス約40km²、メス約20km²（最大100km²超）
- **活動時間帯**：基本的に昼行性
  - 朝4〜7時、夕方5〜9時がピーク
  - 人の生活圏では夜行性に変化
- **GW特有のリスク**：山菜採りの人間と食性が重なり遭遇リスク最大

## 4. ツキノワグマの習性

### 基本性格

- **臆病で人を避ける**：人声や鈴音で通常は逃げる
- **突発遭遇で豹変**：至近距離で出くわすとパニックで攻撃
- 人間を積極的に襲う「捕食型」は極めて稀（だが近年増加傾向）

### 攻撃パターン

- **防御型攻撃**（最も多い）：最初の一撃だけ加えてすぐ離れる
  - 攻撃時間は数十秒〜1分未満
  - 顔面・頭部を狙う
- **子連れ母グマの攻撃**：子グマを守るため、逃げずに向かってくる
  - 5月のタケノコ採りシーズンは親子遭遇リスクが高い
- **食物防衛型**：餌を確保している時に近づくと攻撃

### 身体能力

- **走行速度**：時速40〜50km/h以上（ウサイン・ボルトより速い）
- **木登り**：非常に得意。木に登って逃げるのは無意味
- **嗅覚**：犬の数倍。数km先の食物の匂いを感知
- **水泳**：得意。川や湖は障壁にならない

## 5. クマ遭遇時のステップバイステップ対処法

### 距離別対応マニュアル

#### 50m以上：発見フェーズ

1. **立ち止まる**。走らない、大声を出さない
2. クマの行動を観察する（こちらに気づいているか？）
3. クマに背を向けず、**静かにゆっくり後退**して離れる
4. 風下にいる場合、匂いで気づかれる前に撤退
5. 来た道を戻るか、大きく迂回する

#### 20m前後：警戒フェーズ

1. **穏やかな声で話しかける**（「おーい」など）→ 人間だと認識させる
2. クマが立ち上がった場合 → 威嚇ではなく情報収集中。慌てない
3. **ゆっくり斜め後方に後退**（真正面から下がらない）
4. 熊スプレーの安全装置を外し、いつでも使える状態にする
5. クマが去るのを待つ。去らない場合はじりじり距離を取り続ける

#### 10m以内：緊急フェーズ

1. **熊スプレーを構える**（持っている場合）
2. 5m以内に接近したら**顔面を狙って噴射**
3. スプレーがない場合：
   - **背を向けて走るな**（追跡本能を刺激する）
   - 後ずさりも転倒リスクが高い
4. 突進してきた場合 → **最終防御姿勢**へ移行

#### 突進・攻撃時：防御フェーズ

1. **うつ伏せ**になる
2. **両手を首の後ろで組む**（指を組み合わせてロック）
3. **両ヒジで顔の側面をガード**
4. ザックを背負っている場合は**背負ったまま**うつ伏せに（背中の防御）
5. 足を少し開いて安定させる（ひっくり返されにくくする）
6. **数十秒〜1分間耐える**。クマは防御型攻撃なら一撃で去る
7. クマが離れても**すぐに立ち上がらない**。再攻撃の可能性がある
8. 完全に去ったことを確認してから行動開始

### 秋田大学研究のエビデンス（2025年7月発表）

- 70件の被害を分析
- **防御姿勢をとれた7人全員が重傷なし**
- 姿勢をとれなかった人からは骨折・切断の重傷例が続出
- *Clinical Orthopedic Surgery*誌に掲載された初の科学的検証

### 「死んだふり」の真実

- 昔から言われる「死んだふり」は**半分正しい**
- 正確には「抵抗をやめて急所を隠す防御姿勢をとる」こと
- クマは人を食べるために襲っているのではなく、**自分が逃げるために攻撃**している
- 抵抗しなければ脅威がなくなったと判断して去る可能性が高い
- ただし「捕食型」の攻撃（極めて稀）には無効 → 全力で抵抗・反撃

### 子連れグマの場合

- **最も危険なシナリオ**
- 母グマは子を守るために積極的に攻撃する
- 子グマを見ても**絶対に近づかない**（母グマが必ず近くにいる）
- 子グマと母グマの間に入らない
- 発見したら、来た道をゆっくり戻る
- 5月はGWの時期と子育て期が重なるため、特に警戒

### おすすめ商品比較

|------|---------|---------|-------------|-------------|------|
| **カウンターアソールト CA290 Stronger** | 約10〜12m | 約8〜9秒 | 2% | 24,200円 | 最強クラス。大容量 |
| **カウンターアソールト CA230** | 約9m | 約5〜7秒 | 2% | 19,360円 | 標準モデル。軽量 |
| **SABRE フロンティアーズマン 272mL** | 約12m | — | — | 約15,000円 | 最長射程。霧状散布 |
| **ペッパーマン＋トレーニング ツインパック** | 約4〜5m | 約8秒 | — | 6,930円 | 練習用付き。小型 |

**推奨**：カウンターアソールトCA290。射程・持続時間・濃度のバランスが最良。

### 使い方（手順）

1. **事前準備**：ホルスターに入れ、胸元またはヒップベルトに装着。ザックの中は論外
2. **安全装置解除**：黄色/赤の結束バンドを外す → 白いセイフティークリップを抜く
3. **構え**：本体を片手でしっかり握り、腕を伸ばす
4. **照準**：クマの顔面（目・鼻）を狙う
5. **噴射**：親指でレバーを強く押す。**5m以内**で噴射開始
6. **左右に振る**：霧のカーテンを作るように噴射
7. **退避**：噴射しながらゆっくり後退

### 風向きの注意（最重要）

- **必ず風上に立って噴射**する
- 向かい風の場合、自分が被弾して行動不能になるリスク
- 横風：スプレーが流されてクマに当たらない可能性
- 横風・向かい風では噴射距離が大幅に短くなる
- **最悪のケース**（強い向かい風）：噴射を諦め、防御姿勢に移行
- 自分が少しでも吸うと呼吸困難、目に入ると一時的に失明

### 練習方法

1. **トレーニングスプレーの購入**
   - カールホーネック社「ペッパーマン＋トレーニングスプレー ツインパック」（6,930円）
   - フロンティアーズマン練習用ベアスプレー（Amazon）
   - トレーニングスプレーの中身はほぼ水＋微量アルコール
2. **練習手順**
   - 安全装置の解除を繰り返し練習（素早くできるまで）
   - 屋外で的（木やペットボトル）に向かって噴射
   - 5m先の的に当てる練習
   - **ホルスターからの抜き→解除→噴射**を一連の動作として反復
   - 風向きを変えて練習（横風・微風の条件で感覚を掴む）
3. **所要時間の目安**：パニック下で3秒以内に噴射開始できること

### 保管の注意

- 高温の車内に放置しない（爆発リスク）
- 使用期限を確認（通常3〜4年）
- 飛行機に持ち込めない（現地調達を検討）

## 7. ベアハング（食料吊るし）の具体的やり方

### PCT方式（最も一般的）

#### 必要な装備

- パラコードまたはスペクトラライン：15m以上
- ミニカラビナ：1個
- 食料袋（シルナイロンまたはキューベンファイバー）
- 小さな棒またはペグ（トグル用）
- 重り（石を入れた袋など、ロープ投げ用）

#### 手順

1. **適切な木を選ぶ**
   - 枝の高さ：地上6〜7.5m（20〜25フィート）
   - 枝が幹から1.2m（4フィート）以上突き出している
   - 枝の太さ：ロープを掛ける箇所で直径2.5cm（1インチ）以下
     → クマが枝に乗れない太さであること
2. **ロープを枝に掛ける**
   - ロープの一端に重り（石入り袋）を結ぶ
   - 枝の突端近くに向かって投げる
   - 枝を越えてロープが垂れ下がればOK
3. **食料袋を接続**
   - ロープの一端にカラビナをつけ、食料袋をクリップ
   - ロープのもう一端をカラビナに通す（フリーランニング状態）
4. **引き上げる**
   - ロープを引いて食料袋を枝の位置まで引き上げる
5. **トグルで固定**
   - 手が届く最も高い位置でロープに棒（トグル）をクローブヒッチで結ぶ
   - ロープをゆっくり緩めると、トグルがカラビナに当たって止まる
   - 食料袋は枝とトグルの中間点で吊るされる

#### 配置の数値基準

- **地上からの高さ**：最低3〜3.6m（10〜12フィート）
- **幹からの距離**：最低1.2m（4フィート）
- **テントからの距離**：最低60m（200フィート）
- **風下に設置**

### カウンターバランス方式（代替）

- 1本のロープに2つの袋を同じ重さでぶら下げる
- 両方の袋が枝の両側で同じ高さに吊るされる
- 地面にロープが垂れないのが利点

## 8. テント泊時のクマ対策

### ベアミューダ・トライアングル配置

テント・調理場・食料保管場所を三角形に配置し、それぞれを**最低60〜100m離す**。

```
          食料保管（ベアハング）
              ▲
             / \
            /   \   ← 各辺60〜100m
           /     \
    テント ●-------● 調理場
    （風上）        （風下）
```

### 臭い管理チェックリスト

#### 食料の保管

- [ ] 全ての食料をファスナー付きプラスチックバッグに小分け密封
- [ ] 肉類・魚類は二重袋
- [ ] コーヒー・調味料も密封
- [ ] 食べかけのスナックもテント内に入れない
- [ ] ベアハングまたはベアキャニスターに格納

#### 調理の注意

- [ ] テントから最低60m離れた場所で調理
- [ ] 油はねの少ない調理法を選ぶ
- [ ] 残飯・汁物は絶対に地面に捨てない
- [ ] 食器は調理場で洗い、残り汁は密封して持ち帰る

#### ゴミの管理

- [ ] 空き缶・食品パッケージは全て密封
- [ ] ゴミは食料と同じ場所にベアハング
- [ ] テント周辺にゴミを絶対に置かない

#### 個人の臭い対策

- [ ] 香りの強い歯磨き粉・制汗剤・日焼け止めもベアハングに入れる
- [ ] 調理時に着ていた服はテント外に（匂いがつくため）
- [ ] 食事後に手と顔を拭く

#### テント設置

- [ ] テントは**風上**に設置
- [ ] 見通しの良い場所を選ぶ（クマの接近に気づきやすい）
- [ ] 獣道・川沿い・ベリー茂みの近くは避ける
- [ ] 夜間はテント入口を開けたままにしない

## 9. クマ鈴・ラジオの効果

### クマ鈴

#### 効果がある場面

- 人を恐れる通常のクマに対して有効
- 見通しの悪い道、沢沿い、藪漕ぎでの突発遭遇防止
- 「クマに出会いにくくするもの」であり「絶対に出会わないもの」ではない

#### 効果が薄い場面

- 人に慣れたクマ（「鈴が鳴っても通り過ぎるまで見ているだけ」—猟師の証言）
- 風や沢音で音がかき消される場面
- 餌付け・不法投棄でヒトの食物の味を覚えたクマ（逆に寄ってくる可能性）

#### 推奨

- **消音スイッチ付きの高音質クマ鈴を選ぶ**
- 1つに頼らず、複数の音源を併用

### ラジオ

- 人間の声が流れるため、人の存在を効果的に伝達
- 鈴より広範囲に音が届く
- ただし電池切れリスク

### 総合判断

| 手段 | 有効範囲 | 信頼度 | 備考 |
|------|---------|--------|------|
| クマ鈴 | 20〜50m | 中 | 風・沢音で減衰 |
| ラジオ | 30〜100m | 中〜高 | 電池必要 |
| 声・手拍子 | 20〜50m | 中 | 定期的に出す |
| 熊スプレー | 5〜12m | 高 | 最終手段 |
| 防御姿勢 | 0m | 高 | 科学的に実証済み |

**最も効果的な組み合わせ**：クマ鈴＋定期的な声出し＋熊スプレー携帯

## 10. GWキャンプ行動チェックリスト

### 出発前

- [ ] 熊スプレー購入（カウンターアソールトCA290推奨、24,200円）
- [ ] トレーニングスプレーで練習済み
- [ ] ベアキャニスターまたはベアハング用ロープ＋カラビナ準備
- [ ] ファスナー付きプラスチックバッグ（大量）
- [ ] クマ鈴購入
- [ ] 現地のクマ出没情報を確認（クマダス https://kumadas.net/ ）
- [ ] 岩手県・秋田県のクマ出没マップを確認

### 行動中

- [ ] クマ鈴を常時装着
- [ ] 定期的に声を出す・手を叩く（特に見通しの悪い場所）
- [ ] 熊スプレーはホルスターで胸元に
- [ ] 朝4〜7時、夕方5〜9時は特に警戒
- [ ] 山菜・タケノコの群生地は要注意（クマの餌場）
- [ ] 獣道・糞・爪痕を見つけたら引き返す

### テント泊

- [ ] ベアミューダ・トライアングル配置（60〜100m間隔）
- [ ] 食料は全てベアハングまたはベアキャニスター
- [ ] 臭いの出るものをテント内に入れない
- [ ] 調理用の服は脱いでテント外
- [ ] 就寝前に周囲を確認

### 遭遇時（暗記すべき手順）

1. **止まれ。走るな。**
2. **クマを見ながらゆっくり後退**
3. **穏やかに声を出す**
4. **熊スプレーの安全装置を外す**
5. **5m以内で顔面に噴射**
6. **突進されたら → うつ伏せ → 首の後ろを両手で守る → 耐える**
